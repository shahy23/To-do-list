# 📝 Smart To-Do List (Persistent Web App)

A sleek, functional, and modern To-Do List application. This project focuses on a seamless user experience with a customized aesthetic and data persistence.

## 🚀 Live Demo
Check out the app here: [https://shahy23.github.io/To-do-list/](https://shahy23.github.io/To-do-list/)

## ✨ Key Features
* **Data Persistence:** Tasks are saved in the browser's `LocalStorage`, so they won't disappear even if you refresh the page or close the browser.
* **Modern UI/UX:** Features a custom-designed pink and lavender linear gradient background for a professional look.
* **Full Interactivity:** Easily add new tasks and remove completed ones in real-time.
* **Responsive Design:** Optimized for both desktop and mobile viewing.

## 🛠️ Technologies Used
* **HTML5:** For clean, semantic web structure.
* **CSS3:** Advanced styling using Flexbox and Linear Gradients.
* **JavaScript (ES6):**
    * DOM Manipulation for real-time updates.
    * LocalStorage API for saving user data.
